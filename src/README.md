# Overall structure

```
kk67
|
├── design                      # Prototypes for website
├── assets
│   └── public                  # Public images to be used in website
│
|
├── src
    ├── components              # Reusable components
    ├── constants               # Constant values to be used multiple times
    ├── loaders                 # Loaders for various data to be loaded into website
    │
    ├── router                  # Routing for SPA
    ├── utils                   # Utility functions
    ├── views                   # Views represent the pages that a user will be able to interact ith
    |
    ├── App.vue                 # Parent vue component
    └── main.ts                 # Root component for whole program
```
